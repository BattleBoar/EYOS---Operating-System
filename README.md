## Философия

EYOS — это эксперимент и приглашение к творчеству. Мы верим, что лучшие идеи рождаются в сообществе.

## Сообщество и создатели

Проект создаётся группой энтузиастов. Мы считаем, что каждый, кто вносит вклад — будь то код, идея или даже форк — становится частью истории EYOS. Имена всех, чьи пулл-реквесты будут приняты, а также активных форкеров, пополнят этот раздел.

*   **Основатели:** N.A.C. 
*   **Создатели:** *N_A_C_CEO(Daniil)*
*   **Сообщество:** Вы.

> P.S. Мы смотрим на некоторые 16-битные проекты, написанные на ассемблере, с большим уважением. В них есть особая магия и прямота.

## Дорожная карта (Roadmap)

*   **[Этап 0: Начало]** (предыдущий этап)
    *   Проектирование базовой архитектуры ядра.
    *   Настройка кросс-компилятора и инструментов сборки.
    *   Загрузка в Multiboot-совместимом загрузчике.

*   **[Этап 1: Ядрышко]** (Текущий этап)
    *   Базовая инициализация и обработка прерываний.
    *   Вывод на экран (дружелюбный драйвер терминала :) ).
    *   Управление памятью и менеджер страниц.
    *   64битные команды

*   **[Этап 2: Ядрыще]** (следующий этап)
    *   Динамическое выделение памяти (куча).
    *   Базовая многозадачность
    *   Простейший системный API.
    *   полные 64битные команды

*   **[Этап 3: Свобода]**
    *   Драйверы: ATA, keyboard, Mouse, USB.
    *   Простая файловая система (Ext2).
    *   Портирование пользовательских программ
    *   Поддержка энтузиастов, которые форкнут проект для своих экспериментов.

План может и должен меняться по ходу разработки.

## Технологии и сборка (для Arch Linux)

*   **Язык:** C++ (современные стандарты)
*   **Инструментальная цепочка:** GCC 14, GNU Binutils, GRUB2 + MultiBoot2
*   **Целевая архитектура:** x86, ARM +/-

### Требования для сборки

```bash
sudo pacman -S base-devel gcc git grub qemu qemu-arch-extra xorriso
```
### Клонирование

```bash
   git clone https://github.com/BattleBoar/EYOS---Operating-System.git
   cd EYOS---Operating-System
```
### Компилирование 
``` bash
    ./build86.sh - для x86_64 
    ./buildarm.sh - для arm64
    cd build
```
### Запуск

```bash
   qemu-system-x86_64 -kernel eyos.iso -m 1g
   qemu-system-arm -kernel eyos.iso -m 1g 
```
