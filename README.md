# EYOS — Эксперементальная|Учебная|Узконаправленная операционная система #
> EYOS not UNIX, EYOS not DOS, EYOS is Linux-BSD like.
## Сообщество и создатели

Проект создаётся энтузиастоми. Мы считаем, что каждый, кто вносит вклад — будь то код, идея или даже форк — становится частью EYOS. Имена всех чьи пулл-реквесты будут приняты, а также активных форкеров, пополнят раздел авторов.

*   **Основатели:** N.A.C. 
*   **Создатели:** N_A_C_CEO:(Main Proggrammer C/C++).
*   **Сообщество:** We.

## Дорожная карта

*   **[Этап 0: Начало]** (предыдущий этап)
    *   Проектирование базовой архитектуры ядра.
    *   Настройка кросс-компилятора и инструментов сборки.
    *   Загрузка в Multiboot-совместимом загрузчике.

*   **[Этап 1: Ядрышко]** (Текущий этап)
    *   Базовая инициализация и обработка прерываний.
    *   Вывод на экран.
    *   Управление памятью и менеджер страниц.
    *   64 битные команды

*   **[Этап 2: Ядрыще]** (следующий этап)
    *   Динамическое выделение памяти (куча).
    *   Базовая многозадачность
    *   Простейший системный API.
    *   полные 64 битные команды

*   **[Этап 3: Большая основа]**
    *   Драйверы: ATA, keyboard, Mouse, USB.
    *   Простая файловая система (Ext2-4).
    *   Поддержка arm, 32 битных комманд
    *   Поддержка энтузиастов, которые форкнут проект для своих экспериментов.

План может и должен меняться по ходу разработки.

## Технологии и сборка

*   **Языки:** C++/:23, C/:17, NASM
*   **Инструментальная цепочка:** GNU BinUtils(ld, G++ 11, Nasm, Gcc), GRUB2 + MultiBoot2
*   **Целевая архитектура:** x86_64
*   **Разрядность:** 64bit
*   **Структура ядра:** Монолит, многозадачное, модульное

### Требования для сборки
#### Arch
```bash
sudo pacman -S base-devel nasm gcc git cmake grub qemu xorriso
```

#### Debian
```bash
sudo apt install base-devel nasm gcc git cmake grub qemu xorriso
```

> Советаю включить multilib и настройках

### Клонирование

```bash
   git clone https://github.com/BattleBoar/EYOS---Operating-System.git
   cd EYOS---Operating-System/
```
### Компилирование 
``` bash
    ./build86.sh 
    cd build
```
### Запуск 

```bash
   qemu-system-x86_64 -cdrom eyos.iso -m 1g
```
#### или

```bash
   qemu-system-x86_64 -cdrom eyos.img -m 1g
```
